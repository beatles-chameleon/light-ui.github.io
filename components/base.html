# Light-UI 组件库

Light-UI 组件库旨在丰富CML的社区生态以及组件使用能力。

我们期望通过对CML生态里的组件库不断丰富和完善，为广大开发者提供功能更加齐全的组件，方便所有的开发者快速迭代升级项目。

# 如何使用

  • 首先需要全局安装chameleon-tool
npm i chameleon-tool -g
  • 然后初始化chameleon工程
cml init project
  • 在初始化的chameleon工程中,安装组件库，npm i @cmlkit/light-ui

对于第三方的组件库需要增加babel-path的配置，参考 配置babel-path

const path = require('path');
cml.config.merge({
  babelPath: [path.resolve(__dirname, 'node_modules/test/')],
});
  • 项目中引入组件，根据文档进行使用

以button为例，具体参考 light-button

<script cml-type="json">
{
    "base": {
        "usingComponents": {
          "light-button": "@cmlkit/light-ui/components/light-button/light-button"
        },
        "backgroundTextStyle": "dark"
    }
}
  • 启动项目

在初始化的项目终端执行以下命令便可快速进入开发

cml web dev
